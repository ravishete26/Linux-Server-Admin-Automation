# Linux Server Administration & Automation

## 📌 Objective
To manage and automate Linux servers using shell commands, cron jobs, backups, and monitoring.

## 🛠️ Tools Used
- Linux (Ubuntu/CentOS)
- Shell Scripting
- Cron Jobs
- tar, gzip
- fdisk

## ⚙️ Features
- User & Group Management
- File Permissions
- Package Installation
- Disk Partitioning
- Backup Automation
- Job Scheduling
- System Monitoring
